# Revornix-v0.1.0

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/202506022228803.png)

As an official iteration release, this update brings several important features and optimizations, including but not limited to MCP configuration support.

## Highlights

- Added built-in MCP client support  
- Added several built-in MCP servers, making it easier to interact with your local documents through these clients  
- Optimized parts of the user interaction  
- Added helpful links for easier user access
