# Revornix-v0.6.1

This release brings new features and improvements, making Revornix more powerful and easier to use. Quick tip: the iOS Revornix app is almost finished with its initial development and is expected to launch by the end of this month or early next month.

## Highlights

- Optimized the workflow design and improved concurrency to some extent
- Added the audio document type; you can now analyze your speech through audio documents
- Reworked engines/models into a community mechanism so users can share engines/models
- Improved the user system with identity badges for special users
- Optimized some user interactions (e.g., engine configuration)
- Fixed some known issues
