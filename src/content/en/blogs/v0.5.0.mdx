# Revornix v0.5.0

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/20251117212605932.png)

## Highlights

- Performed **line-level** optimizations across the codebase for a smoother developer experience
- Added extensive error handling to guard against unexpected failures
- Introduced podcast generation so users can create podcasts from documents or sections
- Optimized section sharing and publishing to deliver a polished browsing experience for others
- Refined the overall workflow architecture for documents and sections, greatly reducing unexpected errors while improving traceability
- Added document visibility filtering so currently only documents under published sections or section participants can view the respective content
