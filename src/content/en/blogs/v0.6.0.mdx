# Revornix-v0.6.0

After another month of development, I'm happy to announce that Revornix v0.6.0 is released! This version brings many new features and improvements, making Revornix more powerful and easier to use.

Compared with previous minor iterations, this update includes many new features and optimizations, as follows:

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/20260114204036225.png)

## Highlights

- Backend workflow optimization
- Added manual vector embedding/podcast generation
- Added automatic document tagging
- Added scheduled triggers for column processes
- Refined some UI styles
- Improved some user interactions (e.g., engine configuration)
- Added multimodal capabilities
- Added image generation mechanism
- Added AI process visualization
- Fixed some known issues
