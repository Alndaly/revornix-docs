# Revornix v0.4.2

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/20251019102737489.png)

## Highlights

- Significantly optimized the interaction logic and styling of the column detail page
- Improved the interactions and styling of the column SEO page
- Fixed incorrect merge status displays for column documents
- Refactored the column sharing mechanism, adding user collaboration features and permission controls
- Streamlined the column publishing experience
- Fixed column subscription bugs
- Added edge-case handling to prevent accidental user actions
- Fixed several bugs caused by incorrect permission checks
