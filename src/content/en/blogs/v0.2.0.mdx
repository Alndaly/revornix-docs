## Highlights

- Upgraded some frontend libraries, which improved performance to a certain extent
- Added support for custom document conversion engines; currently supports Jina, Markitdown, and MinerU, so users can choose based on their needs
- Improved document and section search: now you can search by both document title and description
- Refactored markdown conversion engine code for easier future maintenance and expansion
- Fixed calculation errors in statistical charts
