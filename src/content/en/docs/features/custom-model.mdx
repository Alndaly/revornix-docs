# Custom Model

Revornix supports custom third-party models, so you can configure the models you want to use.

<Callout>
Only models compatible with the `openai` protocol are currently supported (i.e., third-party models compatible with the `openai api`).
</Callout>

## How to configure

1. Click the settings button in the top-right to open Settings. Find the model configuration group and click Model Integration, then click Model Config to open the details page.
![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/20251011105845507.png)
2. Click Add Model Provider and configure the basic information. The name and description can be anything for your own reference. Then click Save.
![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/20251011105712488.png)
3. Click Configure Model Group on the provider card to open the model group page. Click Add Model, enter the model name, then click Save on the right. The model name must be the exact official model name; you cannot make it up.
![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/20251011105638316.png)
