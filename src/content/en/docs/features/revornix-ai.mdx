# Revornix AI

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/202505142237714.png)

This project comes with built-in document vectorization and mcp client.

By configuring MCP information in the settings page, you can use all the toolchains provided by the MCP server. For more details, please refer to the [MCP section](./mcp).

At the same time, the project has some built-in MCP Servers (the specific code is in the `mcp-server` directory), you can directly use these built-in MCP Servers, or you can develop your own MCP Server.

Leveraging this feature, we’ve developed AI interactions based on entire documents. You can use Revornix to ask the AI any questions related to your documents. Please ensure that you’ve configured the default Revornix response model in the settings.

Note: Currently, we have not developed a built-in online search feature. If you wish to use online search capabilities, you’ll need to manually configure a model endpoint in Volcano Engine that supports online search. This feature is planned to be integrated into the product via MCP in future updates.
