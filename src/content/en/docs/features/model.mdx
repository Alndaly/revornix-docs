# Custom Model

This system supports custom models, but please note that currently only models compatible with the OpenAI protocol are supported.

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/202507031514832.png)

You can add a custom model by following these steps:

1.	Click the gear icon to enter the settings page.
2.	Locate the model settings section and click the right-hand button to access the model settings page.
3.	Add a model provider and configure the relevant details.
4.	Once the provider is added, click Configure Model Group, add the model, and click Save Button to complete the setup.
