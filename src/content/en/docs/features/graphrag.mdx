# Knowledge Graph

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/20260318192919663.png)

Starting from the `0.4.1-beta` series, the project supports automatic knowledge graph generation.

The process is fully automatic. You only need to add documents to Revornix and the knowledge graph will be generated. It takes some time because Revornix needs to parse documents and build the graph.

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/20260318193016359.png)

You can also interact with the knowledge graph in some simple ways. By clicking on a knowledge point, you can see which articles mention the knowledge point and click to jump to the corresponding article.
