# Knowledge Graph

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/20260115155707731.png)

Starting from the `0.4.1-beta` series, the project supports automatic knowledge graph generation.

The process is fully automatic. You only need to add documents to Revornix and the knowledge graph will be generated. It takes some time because Revornix needs to parse documents and build the graph.
