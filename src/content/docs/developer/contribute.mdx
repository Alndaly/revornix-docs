# 参与贡献

本项目核心代码全部基于我们的[特定协议](https://github.com/Qingyon-AI/Revornix/blob/develop/LICENSE)开源，如果开发者愿意的话，欢迎各位开发者在遵守[协议](https://github.com/Qingyon-AI/Revornix/blob/develop/LICENSE)的基础下参与贡献。

## 如何贡献

### 1. Fork

首先，你需要Fork本项目，然后克隆到本地。

```bash
git clone https://github.com/你的用户名/Revornix.git
```

### 2. 创建分支

在本地创建一个分支，用于开发新功能或修复bug。

```bash
git checkout -b 新分支名
```

### 3. 开发

在本地进行开发，编写代码、修复bug等。

### 4. 提交

将你的修改提交到本地仓库。

```bash
git add .
git commit -m "提交信息"
```

### 5. 推送

将你的修改推送到远程仓库。

```bash
git push origin 新分支名
```

### 6. 创建Pull Request

在GitHub上创建一个Pull Request，申请将你的分支合并到主分支。

### 7. 等待审核

我们将会审核你的Pull Request，如果需要修改，请按照反馈进行修改。

### 8. 合并

如果你的Pull Request合格，我们会将其合并到主分支。

## 注意事项

- 请确保你的代码符合项目的编码规范。
- 请在提交Pull Request之前，先在本地进行测试，确保代码能够正常运行。
