# 专栏

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/202504272033549.png)

设计专栏功能本身是为了能够更方便的集成知识库，方便用户进行知识管理。

同时订阅功能也更好的支持了知识库的传播。

后续将会推出**云版本**，在云版本中会存在付费订阅，此设计的目的是为了提供给知识产出变现，用户可以付费订阅专栏，作者也可以通过订阅获得收益，本平台将会收取 10%的服务费。

## 创建

当前创建专栏的方式仅有一种，即通过网站端创建，暂不支持 API 创建。

### 网站端创建

进入清链应用后，在左侧展开的 sidebar 的专栏两字右侧即有创建入口，点击即可进入创建页面。

#### 配置说明

| 配置项   | 说明                                                             |
| -------- | ---------------------------------------------------------------- |
| 图片     | 每个专栏都必须有一个封面图                                       |
| 专栏名称 | 专栏的名称                                                       |
| 专栏描述 | 专栏的描述                                                       |
| 专栏标签 | 专栏的标签，不仅方便自己筛选，更是能够增加在社区被推荐的概率     |
| 是否公开 | 是否公开，默认私有，如果设为公开，该专栏及其相关文档将在社区可见 |
