# 专栏

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/202504261428007.png)

设计专栏功能本身是为了能够更方便的集成知识库，方便用户进行知识管理。

同时订阅功能也更好的支持了知识库的传播。

后续将会推出**云版本**，在云版本中会存在付费订阅，此设计的目的是为了提供给知识产出变现，用户可以付费订阅专栏，作者也可以通过订阅获得收益，本平台收取10%的服务费。

## 创建

当前创建专栏的方式仅有一种，即通过网站端创建，暂不支持API创建。

### 网站端创建

进入清链应用后，在左侧展开的sidebar的专栏两字右侧即有创建入口，点击即可进入创建页面。

#### 配置说明

- 图片 每个专栏都必须有一个封面图
- 专栏名称 必填项
- 专栏描述 必填项
- 专栏标签 选填，建议填写，不仅方便自己筛选，更是能够增加在社区被推荐的概率
- 是否公开 可选，默认私有，如果设为公开，该专栏及其相关文档将在社区可见
- 是否需要付费订阅 可选，默认不需要，如果设为公开，则用户需要付费订阅才可查看该专栏及其相关文档内容
- 订阅价格 可选

> [!TIP]
> 
> 注意，付费订阅选项和订阅价格选项需要你先开启公开选项才会显示。
