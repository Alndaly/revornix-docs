# Revornix AI

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/202505142237714.png)

本项目内置集成了文档向量化以及MCP客户端。

通过在设置页面内配置MCP相关信息，即可使用MCP服务端提供的所有工具链。详情见[MCP章节](./mcp)。

同时项目还内置了一些MCP Server（具体代码在`mcp-server`目录下），你可以直接使用这些内置的MCP Server，也可以自行开发自己的MCP Server。

基于这一点，我们开发出了可以基于全文档的AI交互，你可以通过Revornix来和AI提问关于文档相关的任何问题，当然，你需要在设置中配置默认的Revornix应答模型。

注意：当前我们并未开发内置的联网搜索功能，如果你想使用联网搜索功能，需要自行配置火山引擎中支持联网搜索的模型端点。（该功能后续将会通过mcp的形式内置在本产品中）
