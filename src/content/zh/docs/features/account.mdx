# 账户体系

由于 Revornix 原生支持多租户，也就是说，你的每个用户都可以拥有自己的账户，并且拥有自己的专栏。

如果你想，Revornix 可以不仅仅是一个你自己的文档库，他可以是一个知识社区，而这也是我真正想最终打造成的样子

**一个高质量的知识社区。**

## 用户主页

每个用户都有一个自己的主页，可以在这里看到 TA 的专栏列表。当前我只设置允许浏览头像、昵称、个性签名、专栏、粉丝以及关注，后续也会视情况进行一定改动。

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/20260115152544556.png)

你可以关注 TA，也可以在 TA 的专栏中留言。

## 账户设置

<Callout>仅支持三方登录的部署版本会有 微信 / Git 等登录方式。</Callout>

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/20260115152656264.png)

为了让社区体系相对完整，我对应的开发了用户的账户设置体系，你可以点击界面左下角的用户头像->账户进入账户设置页面，在这里你可以设置自己的头像、昵称、个性签名。

同时考虑到逻辑正确，我将一些账号相关的设置放在了这个页面，比如你的邮箱更改、密码更改、以及账户 APIKEY。

## 账户 APIKEY

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/20260115152725007.png)

你可以在这里配置你的账户 APIKEY，这个 APIKEY 可以用于你后续的 API 调用，你可以通过这个 APIKEY 来调用三方 API，具体 API 列表详见[API 文档](api)。
