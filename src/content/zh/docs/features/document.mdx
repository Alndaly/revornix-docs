# 文档

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/202504260004883.png)

当前支持速记、网页链接以及文件方式收录文档。

## 文档收录

有多种收录文档、资讯的方式，其中最简单的方式也是最直白的方式就是通过网站端的增加文档。

### 网站端收录

#### 配置说明

- 链接 你想要收录的网站链接
- 标签 你的文档标签，可以添加多个，方便后续筛选查看
- AI 自动总结 开启后，系统会自动调用 AI 接口，将文档内容进行总结，并替换掉默认的网站标题、描述等内容。
- 推送专栏 可以选择你希望推送的专栏，系统会自动将文档推送到你选择的专栏中，并更新该专栏的总结。

### API 收录

当前版本已开放 API 收录文档，你可通过点击应用页面内左下角用户信息->账户进入账户配置页，选择其中的开发 APIKey 配置前往配置。

#### 参数说明

| 参数         | 类型           | 是否必填 | 说明                      |
| ------------ | -------------- | -------- | ------------------------- |
| url          | String         | 否       | 需要收录的文档链接        |
| category     | Integer        | 是       | 0:文件，1:网页，2:速记    |
| from_plat    | String         | 是       | 文档收录来源，请填写`api` |
| auto_summary | Boolean        | 否       | 是否开启 AI 自动总结      |
| sections     | Array[Integer] | 是       | 要推送的专栏列表          |
| labels       | Array[Integer] | 是       | 文档标签列表              |

#### 请求示例

**网站文档**

```python
import httpx
client = httpx.Client()
headers = {
    "API-KEY": "271ed3762ab24444ac2e587add4840f5"
}
response = client.post(
    url="http://localhost:8001/tp/document/create",
    headers=headers,
    json={
        "url": "https://www.baidu.com",
        "category": 1,
        "from_plat": "api",
        "auto_summary": True,
        "sections": [],
        "labels": [],
    },
)
```

#### 响应示例：

```json
{ "document_id": 26 }
```
