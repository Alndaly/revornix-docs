# 文档转化引擎

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/202506181320196.png)

在`v0.2.0`及之后的版本里，文档转化引擎将可以由用户自行决定，而不再局限于MinerU（文件）/Jina（网站）。具体配置页面入口在**设置->引擎管理**。

目前官方所提供的包含三种引擎，分别是

- MinerU
- Jina
- Markitdown

<Callout type="warning">
当前Jina引擎还不支持文件文档转化，仅支持网站文档转化，所以如果是文件类型的请设置为 MinerU / Markitdown。
</Callout>

点击对应引擎安装后即可进行对应配置，配置完成后即可使用对应引擎进行文档转化。

注意对应的配置项在你点击已安装的引擎的配置按钮后会在弹窗下方显示，可根据范例设置参数。
