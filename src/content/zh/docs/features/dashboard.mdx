# 仪表盘

整个系统中，大部分的核心基础信息均在仪表盘页面能够用非常快捷的方式查看。我也会在后续的开发中，逐步将更多的内容通过合理的逻辑以及UI布局迁移到仪表盘页面。

如果你是**新用户**注册，你也会在这个仪表盘页面的顶部看到待办事项，比如模型绑定、引擎配置等等。

<Callout>
由于数据是实时的，任何文档一经删除，将不会被记入统计范畴内，也就不会在这个页面内的数据中有所体现。
</Callout>

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/20251011141251012.png)

当前包含的数据范畴如下

- 当日总结（即当天所有文档总结后生成的专栏）
- 当前热搜集合（基于[daily-hot](./hot-search)服务）
- 按日统计的文档收集情况
- 当前有多少未读文档，以及最近的一次收集但仍旧未读的文档
- 最近阅读文档统计，以及最近一个阅读的文档
- 收藏文档统计，以及最近的一次收藏
- 文档和专栏的标签
