# 仪表盘

整个系统中，部分的核心基础信息均被我内迁到了仪表盘页面，你可以通过仪表盘页面方便快速了解自己的当前知识面情况。我也会在后续的开发中，逐步将更多的内容通过合理的逻辑以及UI布局迁移到仪表盘页面。

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/202507292031988.png)

当前包含的数据范畴如下

- 按日统计的文档收集情况
- 当前有多少未读文档
- 最近阅读文档累计
- 收藏了多少文档
- 文档和专栏的标签

<Callout>
由于数据是实时的，任何文档一经删除，将不会被记入统计范畴内，也就不会在这个页面内的数据中有所体现。
</Callout>
