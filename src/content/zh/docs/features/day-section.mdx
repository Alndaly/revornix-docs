# 今日专栏

为了将知识日期化整理，便于用户进行知识回顾，我设置了这么一个专栏。

<Callout>
注意，目前没有办法取消今日专栏对创建文档的引入，即你所有创建的文档都会被自动添加到今日专栏中。
同时该类目专栏默认自动开启自动插图，并且其专栏进程触发机制默认为时间早上6点和晚上6点（你可以自己修改）。
</Callout>

这是一个特定的专栏类型，不需要用户手动增加配置，系统会根据你的文档收集情况自动更新。

即：每当用户于新的一天创建第一份文档的时候，系统会自动创建一个今日专栏，并将文档添加到今日专栏中，然后进行总结分析。

该功能和**通知功能**是有一定绑定的，我也特地为当日总结做了一份通知模版，你可以在通知任务页面内直接配置该模版，这样用户在自己设定的时间点就会收到当天的总结。
