# RSS 订阅

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/202508311049845.png)

由于有时候用户还存在着跟踪一个网页的更新的需求，比如博客或者up主的视频、推文等，所以支持 RSS 订阅是一个很自然的需求。

自`0.4`系列版本以后，系统内置了RSS订阅功能，当前你可以通过在左侧菜单栏中选RSS子项来快速配置增加RSS订阅源，通过该方式，理论上你可以订阅所有网站的更新。

同时系统内嵌了[RSSHub](https://docs.rsshub.app/zh/)作为RSS订阅源的生成器，所有该项目支持的订阅源Revornix可无缝接入。

如果你有一些网站还想增加RSS订阅源，但是RSSHub不支持，你可以基于RSSHub的[官网文档](https://docs.rsshub.app/zh/joinus/)来进行编写增加。
