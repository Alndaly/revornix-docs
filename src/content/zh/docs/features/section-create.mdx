# 专栏创建

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/20251019124934212.png)

普通专栏的创建极其简单，只需要点击左侧菜单栏的专栏右侧的创建即可。

而针对于特定的专栏，比如今日专栏，是不需要用户进行手动创建的，在你增加文章的同时，系统会自动为你创建今日专栏，并在后续增加文章过程中，系统会自动根据原总结和新的文档生成新的专栏。

<Callout>针对今日专栏，目前没有关闭的选项，即任何加入的文档都会自动被并入今日专栏并进行分析总结。</Callout>

同时我还提供了API方式来创建专栏，具体可以参考[API文档专栏创建接口](./api/#tpsectioncreate)，当然，npm库和pip库中的revornix包也具有相应的功能，具体可以参考对应的github开源网站文档：[Revornix-Npm-Lib](https://github.com/Qingyon-AI/Revornix-Npm-Lib)，[Revornix-Python-Lib](https://github.com/Qingyon-AI/Revornix-Python-Lib)。
