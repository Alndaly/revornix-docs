# 知识图谱

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/20260318192919663.png)

自`0.4.1-beta`系列开始，本项目支持自动生成知识图谱。

过程全自动，你只需要向Revornix增加文档，知识图谱就会自动生成。当然，这需要一些时间，因为Revornix需要解析文档并生成知识图谱。

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/20260318193016359.png)

你可以和知识图谱进行一些简单的交互，通过点击知识点，你可以看到有哪些文章中提到了该知识，并且点击跳转到对应的文章内。
