# 隐私策略

本项目是一个开源项目，旨在为用户提供信息的管理、搜集与分析。我们非常重视用户的隐私与数据安全。

### 数据收集

默认情况下，我们不会收集任何用户的个人信息。该应用在本地运行，不会主动上传、分享或储存用户数据。

### 使用权限

如果本项目请求以下权限或系统功能，仅用于其对应的功能实现：

- 网络访问：用于加载远程内容或 API 数据。
- 本地存储访问：用于保存用户的设置或数据。
- 通知权限：用于向用户推送提醒消息，不会用于收集信息。
- 麦克风/摄像头/位置等权限（如适用）：仅用于应用特定功能（如音视频传输），不会在后台采集保存。

### 第三方服务

如本项目依赖第三方服务（例如 Firebase、Apple API 等），请参考对应服务的隐私政策。我们不会主动向第三方发送任何未经用户同意的个人信息。

### 本地数据存储

所有用户数据默认存储于本地设备。用户可以自由导出、修改或删除自己的数据。

### 开源说明

本项目为开源项目，任何人都可以查看、修改或部署代码。我们鼓励用户自行部署并根据需要调整隐私相关实现。
