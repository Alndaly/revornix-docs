# 隐私协议

Revornix旨在为用户提供完美并且丝滑的信息的搜集、分析与分享流程。我们非常重视用户的隐私与数据安全。由于同时具备开源版本和云版本，所以这里会有两份隐私协议。

> 我们承诺Revornix将永远保持开源，你可以选择**自行部署**的方式来免费使用Revornix几乎所有功能。

## Revornix开源版本

### 数据收集

作为开源软件平台提供方，我们不会收集你方任何用户的个人信息，你所有的数据都由你自己掌控，自己负责。

同时，若你的用户信息遭到泄漏，我们也不承担任何责任。

### 使用权限

参照项目功能，如果本项目请求以下权限或系统功能，仅用于其对应的功能实现：

- 网络访问：用于加载远程内容或 API 请求。
- 本地存储访问：用于保存或者读取用户的自定义设置项、文档、专栏等数据。
- 通知权限：用于向用户推送提醒消息，我们承诺非必要不进行通知，且基于Revornix的通知解耦设计，用户可以自行决定接受哪些通知，甚至可以自行配置通知源和目标。

### 第三方服务

本项目依赖一些第三方服务（例如 模型提供方，Apple API 等），请参考对应服务的隐私政策。我们非必要不会主动向第三方发送任何未经用户同意的个人信息。同时注意用户使用云大模型时，对话是不可避免会发送给模型提供方的，该类隐私问题具体协议请参考模型提供方。用户或者经授权后的Revornix分发商有义务了解并遵守这些隐私政策。

### 开源说明

本项目为开源项目，任何人都可以查看、修改或部署代码。我们鼓励用户自行部署并根据需要调整隐私相关实现。

## Revornix官方云版本

该版本请认准一级根域名revornix.com，非该根域名皆不是Revornix官方提供，请谨慎使用。非官方认可授权营运的版本，我们不会提供任何隐私保护。

### 数据收集

作为云服务提供方，我们仅收集以下必要隐私信息：

- 用户邮箱：用于注册、登录、找回密码等。
- 用户IP地址：用于统计用户访问量，以及防止产品被滥用。
- 用户设备信息：用于统计用户访问量，以及通知能力。
- 用户行为数据：用于统计用户访问量。
- 用户反馈数据：用于改进产品。
- 用户微信信息：用于微信登录、微信支付等。
- 用户Google信息：用于Google登录等。
- 用户Github信息：用于Github登录等。
- 用户手机号：用于手机登录等。
