# Revornix-v0.1.0

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/202506022228803.png)

作为一个正式迭代版本，本次更新带来了一些比较重要的功能和优化，包括但不限于MCP的配置。

## 更新亮点

- 增加了内置MCP客户端支持
- 增加了一些内置MCP服务端，可以方便的通过这些客户端和你本地的文档互动
- 优化了部分用户交互
- 补充了一些帮助链接方便用户查看
