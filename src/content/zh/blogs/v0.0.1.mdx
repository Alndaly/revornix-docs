# Revornix-v0.0.1

这是项目成立以来的第一个正式发布版本。肯定会存在诸多 bug，感谢大家的耐心和理解。我们会持续优化和改进。

## 更新亮点

1. 支持链接文档、快速笔记和文件文档
2. 自动将文件文档转换为 Markdown 格式
3. 支持文档搜索、分类和标签
4. 全文搜索
5. Markdown 预览
6. 多用户支持
7. 暗黑模式
8. 多语言支持
9. 热搜集合
10. 栏目（分类/主题）管理
11. Revornix AI 问答，基于文档
12. 集成网页搜索的 AI 回答
13. 每日总结
14. 第三方 API 集成
