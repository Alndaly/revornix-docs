# Revornix-v0.0.3

经过这几天的努力，我们非常高兴地宣布 Revornix v0.0.3 版本的发布！这个版本带来了许多改进和修复，以提升您的体验。

更新亮点

- 增加了默认Revornix AI应答模型和文档阅读模型的切换机制，现在你可以在应用内自由选择模型而不需要在环境变量中预先定义了！
- 修复了大量已知问题包括但不限于ai总结、热搜集结、docker build工作流等等。
